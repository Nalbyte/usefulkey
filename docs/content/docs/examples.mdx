There are a few examples available showing how to integrate UsefulKey with various frameworks and runtimes.
## Available Examples

- **[Next.js (App Router)](https://github.com/nalbyte/usefulkey/tree/main/examples/nextjs)** - Full-stack Next.js application with API routes
- **[Astro](https://github.com/nalbyte/usefulkey/tree/main/examples/astro)** - Server-side Astro application with API endpoints
- **[Hono](https://github.com/nalbyte/usefulkey/tree/main/examples/hono)** - Lightweight Hono server implementation
- **[Bun](https://github.com/nalbyte/usefulkey/tree/main/examples/bun)** - High-performance Bun runtime server
- **[Cloudflare Workers](https://github.com/nalbyte/usefulkey/tree/main/examples/cloudflare-worker)** - Edge deployment with Cloudflare D1 and KV

All examples implement the same core functionality:
- API key creation and management
- Key verification with rate limiting
- Health check endpoints
- Proper error handling and security

Visit the [GitHub repository](https://github.com/nalbyte/usefulkey) for complete setup instructions and deployment guides.
